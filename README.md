# Juego de ahorcado en JAVA
Creamos un juego ahorcado en java en donde se pueda ingresar por letras para adivinar la palabra, se tienen 10 intentos. Se gana al completar la palabra antes de gastar los 10 intentos.

## Herramientas utilizadas
1. Clase scanner
2. Declaraciones y asignaciones
3. Arreglos
4. Estructura de control: Iterativa (Bucle)
5. Clase String y sus metodos valueOf, chartAt y length()
6. Clase Character y su metodo toLowerCase
7. Estructura de control: condicional
